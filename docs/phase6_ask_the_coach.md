# Phase 6 — Ask the Coach

Ask the Coach is the professional LinkedIn differentiator module.

It answers strategic questions using:
- 48-team tournament simulation
- knockout path estimation
- dark-horse detection
- what-if sensitivity analysis
- optional Mistral Chief Analyst polishing

Example questions:
- What is Portugal's most likely path to the final?
- Which team is the biggest dark horse?
- What happens if Mbappé misses the quarter-finals?

The module runs locally and does not require Mistral. With `MISTRAL_API_KEY`, it upgrades the deterministic answer into a polished coach-style briefing.
